# Rhea Image Creation Skill

A master-class guide for generating visual assets for the Rhea and Eidos AGI ecosystem. Enforces the **Language as Momentum** discipline—highly verbose, technically precise prompts that produce sovereign, minimalist imagery.

## 🏛️ Core Philosophy: Language as Momentum
Image models are momentum systems. Every additional word in a prompt narrows the manifold of possible outputs, pulling the result away from generic AI centroids and toward the **Eidos AGI aesthetic**.

**The Rule**: If your prompt is shorter than four sentences, it is insufficient. Specify hex codes, line weights, compositional rules, and explicit negative constraints.

---

## 🎨 The Midnight Palette
These are the only colors permitted in primary Rhea assets. Always specify them by hex code.

| Role | Hex | Use |
| :--- | :--- | :--- |
| **Background** | `#0a0a14` | The mandatory canvas. Near-black with a violet bias. |
| **Primary Accent** | `#6c8aff` | Electric blue. Focal geometry, active nodes. |
| **Secondary Accent** | `#a78bfa` | Slate purple. Supporting curves, depth tiers. |
| **Text / Lines** | `#c8c8d8` | Technical silver. Labels, axis ticks, structural lines. |

## 📐 Prompt Scaffold (The Verbosity Mandate)
Every image prompt must include these six elements in order:

1.  **Domain Frame**: Establish the context (e.g., "A technical system architecture diagram from a systems engineering paper").
2.  **Subject**: Name the subject concretely using structural language (e.g., "A three-node fallback mesh").
3.  **Composition**: Define layout and projection (e.g., "Orthographic projection, center-weighted with 15% margin").
4.  **Palette Directive**: Assign hex codes to specific roles.
5.  **Line Discipline**: Specify weights (e.g., "1.2px silver structural lines, 2px blue focal lines").
6.  **Negative Constraints**: Explicitly forbid clichés (e.g., "No robots, no glowing brains, no lens flare, no 3D rendering").

---

## 🚫 Prohibited Tropes
- **No literal AI imagery**: No robots, androids, or human figures.
- **No "Matrix" clichés**: No falling code, glowing orbs, or neon grids.
- **No decorative clutter**: No lens flares, depth-of-field blur, or motion blur.
- **No "Workshop" elements**: No steampunk, messy textures, or warm amber lighting. Rhea is a sovereign momentum engine, not a grounded workshop.

## 🛠️ Operating Discipline
1.  **Never ship the first generation**. Use the first result as a baseline, analyze the drift, and double the prompt verbosity for the second round.
2.  **Anchor with hex codes**. The model reacts more accurately to `#0a0a14` than to "dark blue."
3.  **Subtract, don't add**. If an image is too busy, do not add more descriptions; add more negative constraints.
